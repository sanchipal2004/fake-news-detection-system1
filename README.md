#  Fake News Detection System

> An AI-powered web application that detects whether a news statement is **Real or Fake** using  Natural Language Processing (NLP) and Machine Learning models.

---

##  Overview

The **Fake News Detection System** is a full-stack web application that helps users verify the authenticity of news articles or statements.  
It uses a **trained Machine Learning model** (e.g., Logistic Regression,  BERT) to classify text as *Fake* or *Real* based on its linguistic features.

Users can simply paste any news headline or article text into the input box and get an instant prediction along with confidence scores.

---

##  Key Features

-  **Fake News Detection:** Uses NLP-based ML models to classify text as Real or Fake.  
- **Confidence Score:** Displays the prediction confidence in percentage.    
- **RESTful API Integration:** Backend API built using Node.js and Express connects with the ML model.  
- **Modern Frontend:** Developed using React.js with TailwindCSS for a clean and responsive UI.  
- **Real-time Analysis:** Processes user input instantly and displays the result dynamically.
